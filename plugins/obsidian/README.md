# Obsidian

Claude Code plugin for working with [Obsidian](https://obsidian.md) vaults using the Obsidian CLI. Includes vault organization context and an LLM-maintained knowledge base.

## Skills

| Skill | Description |
|-------|-------------|
| `obsidian-cli` | Obsidian CLI reference — notes, tags, tasks, properties, plugins, sync, and more |
| `obsidian-notes` | Vault organization context — reads `Vault Organization.md` and `Frontmatter Conventions.md` from the vault |
| `obsidian-knowledge-base` | LLM-maintained knowledge base — reads `Knowledge Base.md` from the vault for conventions and workflows |
